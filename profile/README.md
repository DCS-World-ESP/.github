## ‚ö° Setup Guide

1. **Download** the ESP-enabled package.
2. Place files inside your DCS World root folder.
3. Run `ESPOverlay.exe` as administrator.
4. Launch DCS World.
5. Press `INSERT` to open the overlay.

```ini
[esp]
aircraft_color=yellow
ally_color=blue
ground_units=green
missile=red
distance=true
```
